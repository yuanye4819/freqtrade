## 1. 一句话介绍

> 自动帮你买卖加密货币的机器人。它只在**趋势明确、价格极端、放量反弹**的时候出手，然后用多层保护确保不亏大钱。

**做多** = 在牛市的回调中抄底  
**做空** = 在熊市的反弹中做空

## 4. 它是怎么买卖的（核心逻辑）

### 做多逻辑（上升趋势中抄底）

```
市场在涨（价格 > 200EMA）
      ↓
突然回调下跌
      ↓
RSI 跌到 26 以下（严重超卖）
      ↓
价格碰到布林带下轨（极其便宜）
      ↓
成交量放大（有人抢着抄底）
      ↓
反弹开始，RSI 穿过 26 → 触发买入
      ↓
持有，等它慢慢涨
```

### 做空逻辑（下降趋势中做空）

```
市场在跌（价格 < 200EMA）
      ↓
突然反弹上涨
      ↓
RSI 涨到 76 以上（严重超买）
      ↓
价格碰到布林带上轨（极其昂贵）
      ↓
成交量放大（有人抢着跑）
      ↓
反弹结束，RSI 跌穿 76 → 触发做空
      ↓
持有，等它慢慢跌
```

### 七个入场条件

| # | 条件 | 做多时的含义 | 做空时的含义 |
|---|------|-------------|-------------|
| 1 | 价格 vs 200EMA | 价格在上面（涨） | 价格在下面（跌） |
| 2 | RSI 交叉 | 从超卖反弹 | 从超买回落 |
| 3 | 布林带位置 | 在下轨附近（便宜） | 在上轨附近（贵） |
| 4 | 成交量 | 放量（有人抄底） | 放量（有人出逃） |
| 5 | ADX 趋势强度 | >31（强趋势） | >31（强趋势） |
| 6 | 1h 趋势确认 | 1h EMA50 > EMA200 | 1h EMA50 < EMA200 |
| 7 | K线有效 | 成交量>0 | 成交量>0 |

## 5. 出场保护机制

策略有五层保护，从最优先到最终防线：

### 第一层：ROI 止盈

| 持仓时间 | 收益要求 | 逻辑 |
|----------|----------|------|
| 30 分钟 | +1% | 短线赚够就跑 |
| 60 分钟 | +2% | 拿了1小时该见分晓了 |
| 120 分钟 | +3% | 拿得越久要求越高 |
| 无条件 | +0.5% | 防止无限持仓 |

### 第二层：追踪止损

```
盈利达到 1.9% 后启动
从最高点回撤 3.3% 触发卖出

例：买入 100 → 涨到 110（+10%）
    → 回落到 106.7（回撤 3.3%）
    → 自动卖出，锁住 +6.7%
```

### 第三层：动态止损

```
赚得越多 → 守得越紧

盈利 > 5%  → 止损提到 +3%（锁住大部分利润）
盈利 > 3%  → 止损提到 +1%
盈利 > 2%  → 止损提到开仓价（保本出）
盈利 > 1%  → 止损缩到 -1.5%（少亏）
```

### 第四层：硬止损

```
亏损达到 -4%，无条件割肉
目前 20 笔交易中 0 次触发
```

### 第五层：趋势破位

```
价格穿破 200EMA → 趋势可能反转 → 清仓离场
```

## 6. 所有参数说明

### 做多参数

| 参数 | 默认值 | 范围 | 作用 | 调大效果 | 调小效果 |
|------|--------|------|------|----------|----------|
| `buy_rsi` | 26 | 15-40 | RSI 阈值 | 更容易买 | 更难买 |
| `buy_bb` | 0.17 | 0.1-0.5 | 布林带位置 | 更容易买 | 更难买 |
| `buy_adx` | 31 | 15-40 | 趋势强度 | 更难买 | 更容易买 |

### 做空参数

| 参数 | 默认值 | 范围 | 作用 |
|------|--------|------|------|
| `sell_rsi` | 76 | 60-85 | RSI 做空阈值 |
| `sell_bb` | 0.80 | 0.5-0.9 | 布林带做空位置 |

| 参数 | 默认值 | 范围 | 作用 |
|------|--------|------|------|

## 7. 如何运行

```powershell
cd D:\freqtrade
.\freqtrade-env\Scripts\freqtrade.exe backtesting `
  --config config.json `
  --strategy Strategy1HyperV3 `
  --timerange 20250101- `
  --data-format-ohlcv feather
```

```powershell
cd D:\freqtrade
.\freqtrade-env\Scripts\freqtrade.exe hyperopt `
  --config config.json `
  --strategy Strategy1HyperV3 `
  --timerange 20250101-20250601 `
  --spaces buy sell `
  --epochs 200 `
  --hyperopt-loss SharpeHyperOptLossDaily `
```

## 8. 如何修改

### 增加交易频率

在 `Strategy1HyperV3.0.py` 中找到这段并修改：

```python
# 放宽做多条件 → 更多交易
buy_rsi = IntParameter(15, 40, default=30, space="buy")   # 26→30
buy_bb  = DecimalParameter(0.1, 0.5, default=0.30, decimals=2, space="buy")  # 0.17→0.30
buy_adx = IntParameter(15, 40, default=25, space="buy")   # 31→25

# 放宽做空条件 → 更多做空机会
sell_rsi = IntParameter(60, 85, default=70, space="sell")  # 76→70
sell_bb  = DecimalParameter(0.5, 0.9, default=0.75, decimals=2, space="sell")  # 0.80→0.75
```

### 只做多不做空

```python
can_short: bool = False   # True → False
```

### 调紧止损

```python
stoploss = -0.03   # 最多亏3%而不是4%
```

在 `config.json` 中：

```json
    "OKB/USDT:USDT",
    "SOL/USDT:USDT"
```

## 10. 常见问题

### Q: 为什么交易这么少？

A: 设计如此。七个条件相当严格，宁可不交易也不乱交易。想在 18 个月里更多交易→参考第 8 节放宽参数。

### Q: 为什么回撤这么低？

A: 因为入场条件严格（只在高胜率时机出手）+ 多层止损保护（赚了就锁利润，亏了就快跑）。

### Q: 做空和做多哪个赚得多？

A: 做空 11 笔赚了 +3.77%，做多 9 笔赚了 +2.19%。但样本太少，不能下结论。

### Q: 代理关了怎么办？

A: 代理必须开着（127.0.0.1:7890），否则无法连接 OKX 交易所。

### Q: 能不能实盘？

A: 理论上可以，但建议先用模拟盘跑至少 3 个月，确认表现符合预期后再考虑。

### Q: 这个策略能保证赚钱吗？

A: **不能。** 历史表现 ≠ 未来收益。市场永远在变，任何策略都可能失效。
